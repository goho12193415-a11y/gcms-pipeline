一、系统要求
-----------
1. Windows 10 或 Windows 11
2. Python 3.11 或更高版本
3. 约 2GB 磁盘空间（软件 3.5MB + NIST 谱库 807MB + Python 环境约 1GB）


二、安装 Python（如已安装可跳过）
---------------------------------
方法一：Microsoft Store 安装（推荐，无需管理员权限）
  1. 打开 Microsoft Store，搜索 "Python 3.11"
  2. 点击安装
  3. 安装完成后，打开命令提示符（Win+R，输入 cmd，回车）
  4. 输入 python --version，显示版本号即安装成功

方法二：官网安装
  1. 访问 https://www.python.org/downloads/
  2. 下载 Python 3.11.x 安装包
  3. 安装时务必勾选 "Add Python to PATH"
  4. 完成后验证同上


三、安装依赖包（只需执行一次）
-----------------------------
1. 双击本文件夹中的 "安装依赖.bat"
2. 等待安装完成（约 2-5 分钟，视网络速度而定）
3. 出现 "Setup complete!" 即安装成功

如安装失败，可手动执行：
  Win+R → 输入 cmd → 回车
  输入：pip install numpy scipy pandas openpyxl pymzml pyms-nist-search


四、确认文件夹结构
-----------------
安装完成后，文件夹结构应如下：

  GCMS_Software/
  ├── 安装依赖.bat
  ├── 安装教程.txt        ← 本文件
  ├── 使用教程.txt
  ├── 谱库/               ← NIST 谱库（807MB，已就绪）
  │   ├── mainlib/
  │   └── nist_ri/
  └── 软件/               ← 软件本体
      ├── 启动.bat          ← 日常使用双击此文件
      ├── gui.py
      ├── pipeline.py
      ├── library/          ← RI 数据库
      └── ...

请勿移动或重命名 "谱库" 和 "软件" 文件夹，否则软件无法找到 NIST 库。


五、启动软件
-----------
双击 "软件\启动.bat" 即可打开图形界面。

如窗口一闪而过，说明依赖未正确安装，请重新执行第三步。


六、卸载
-------
直接删除整个 GCMS_Software 文件夹即可，无注册表残留。


七、常见问题
-----------
Q: 启动报错 "No module named 'xxx'"
A: 依赖包未完整安装，重新执行 "安装依赖.bat"

Q: 启动后提示找不到 NIST 库
A: 确认 "谱库" 文件夹与 "软件" 文件夹在同一目录下

Q: 处理 .RAW 文件报错
A: 需额外安装 ThermoRawFileParser
   下载：https://github.com/compomics/ThermoRawFileParser/releases
   解压后将 ThermoRawFileParser.dll 所在目录添加到系统 PATH

Q: 软件是否收费？
A: 完全免费。MIT 开源协议。
